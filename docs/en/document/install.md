# Installation

## pnpm

Install using the fast and disk-space efficient package manager [pnpm](https://pnpm.io/zh)

```shell
pnpm add --save-dev dk-plus
```

## npm

Install using `npm`

```shell
npm install --save-dev dk-plus
```

## yarn

Alternatively, install using `yarn` 

```shell
yarn add --save-dev dk-plus
```
